🎵 오구톡 알람음 디자이너

프리셋 선택 → 미리듣기 → 마음에 들면 다운로드. Claude에게 새 톤 부탁해서 붙여넣기도 가능.

1️⃣ 프리셋 선택 (클릭 시 아래 편집창에 로드)

2️⃣ 톤 설정 (Claude 결과를 여기에 붙여넣기)

3️⃣ 재생 옵션

4️⃣ 재생 / 다운로드

버튼을 눌러 시작하세요.

💡 Claude에게 새 톤 부탁하기

Claude 채팅에 이렇게 말하세요:
"오구톡 알람음 [느낌] 스타일로 만들어줘. JSON 형식으로."

예시:
• "부드러운 일본 만화 분위기"
• "새 지저귀는 듯한 휘파람"
• "명상 종소리 길게, 2초 정도"
• "슈퍼마리오 동전 줍기 같은 명랑한 톤"

Claude가 만들어준 JSON을 위 편집창(②번)에 붙여넣고 미리듣기 → 마음에 들면 다운로드.
📖 JSON 형식 설명 (Claude에게 알려줄 때 사용) 
{
  "name": "톤 이름",
  "duration": 0.7,         // 톤 전체 길이 (초)
  "notes": [
    {
      "freq": 880,          // 주파수 (Hz). 880=A5, 523=C5, 1047=C6
      "start": 0,           // 시작 시점 (초)
      "dur": 0.13,          // 이 음의 길이 (초)
      "type": "sine",       // 파형: sine(부드러움) / triangle(따뜻) / square(디지털) / sawtooth(거침)
      "gain": 0.25,         // 볼륨 (0~1, 보통 0.1~0.3)
      "freqEnd": 1320,      // (선택) 슬라이드 끝 주파수 — 휘파람/사이렌 효과
      "fadeIn": 0.02        // (선택) 페이드인 시간 — 부드러운 시작
    }
  ]
}

음역 참고:
  C4=261, C5=523, C6=1047
  E4=329, E5=659, E6=1319
  G4=392, G5=784, G6=1568
  A4=440, A5=880, A6=1760
  D5=587, F5=698, B5=988

디자인 팁:
  • sine: 부드럽고 따뜻 (오구 느낌)
  • triangle: 풍성하면서 부드러움
  • square: 8비트 게임/디지털 느낌
  • 높은 freq(>1500): 밝고 주목성 ↑
  • 낮은 freq(<500): 따뜻하고 진중함
  • gain은 동시 발음 수에 따라 조정 (총합 0.5 이하 권장)
