# PRD — 오구톡 (OguTalk)

## 1. 제품 개요

**오구톡**은 매시 **59분**에 짧은 "오구" 사운드와 함께 알림을 울려 스마트폰 과몰입을 방지하고 시간 감각을 되찾게 해주는 모바일 앱이다. 단순한 타이머를 넘어 경제 트렌드 정보·목표 관리·할일 추적을 한 곳에 묶어 "알림이 울릴 때마다 하루를 점검"하는 루틴을 만든다.

## 2. 타겟 사용자

| 세그먼트 | 설명 |
|---|---|
| 주요 타겟 | 스마트폰에 몰입해 시간 감각을 잃는 직장인·학생 (20~40대) |
| 서브 타겟 | 경제 뉴스·주식 정보를 빠르게 파악하고 싶은 재테크 관심층 |

## 3. 핵심 기능

### 3-1. 59분 알람 (코어)
- 매시 59분 0초에 Web Audio API로 오구 사운드 재생 (C5→E5 두 음, 0.5초)
- 브라우저 Notification API로 푸시 알림 발송
- 알람 팝업: 명언 + 경제상식 + 미완료 할일 수 표시
- 발동 후 몰입 시간(immersionMinutes) 초기화
- 시간대별 ON/OFF 설정 (07~23시 기본)

### 3-2. 목표 관리
- 연간 → 월간 → 주간 → 일간 4단계 계층 구조
- 진행률 바(%), 달성/미달성 토글
- Supabase `goals` 테이블에 실시간 동기화

### 3-3. 할일 (Todos)
- 빠른 할일 추가·완료·삭제
- 알람 팝업에서 미완료 건수 실시간 표시
- Supabase `todos` 테이블에 실시간 동기화

### 3-4. 키워드 & 트렌딩 분석
- **트렌딩 탭**: 네이버 데이터랩 실시간 검색어 트렌드 (경제/AI/사회/글로벌 4개 카테고리 × 주간·월간)
- **내 키워드 탭**: 관심 키워드 등록 → 클릭 시 네이버 뉴스 실시간 검색 결과 표시
- CORS 우회: Supabase Edge Function 프록시 사용
- 프리미엄 플랜 전용 기능

### 3-5. 경제 상식 카드
- Supabase `economic_tips` 테이블 (초기 10개 입력)
- 알람 팝업·홈 화면에서 랜덤 표시

### 3-6. 인증
- 이메일/비밀번호 로그인·회원가입 (Supabase Auth)
- 카카오 OAuth (Supabase signInWithOAuth → PKCE 콜백)
- 가입 시 `profiles`, `alarm_settings`, `user_preferences` 자동 생성 (DB 트리거)

## 4. 수익 모델

| 플랜 | 가격 | 제공 기능 |
|---|---|---|
| 무료 | 0원 | 알람, 할일, 목표 관리 |
| 프리미엄 | 월 2,900원 | + 키워드 분석, 주식 정보, 광고 제거 |

## 5. 비기능 요구사항

- **플랫폼**: PWA (Progressive Web App) — 안드로이드·iOS 홈 화면 설치 가능
- **최대 너비**: 420px (모바일 우선)
- **테마**: 다크 테마 고정 (#0f172a 배경)
- **오프라인**: Workbox 캐시로 기본 셸 오프라인 제공
- **배포**: Vercel (자동 CD, GitHub push 트리거)
